# Sentiment Analysis App

This is a Python mini-project for sentiment analysis.

## Files

- `sentiment_app.py` - main app
- `test_sentiment_app.py` - unit tests
- `blocked_words.json` - blocked words list

## How to run

1. Create a virtual environment: `python -m venv venv`
2. Activate the environment and install dependencies
3. Run `sentiment_app.py`
