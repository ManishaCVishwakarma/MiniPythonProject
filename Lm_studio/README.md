# LM Studio RAG Project

This module demonstrates a Retrieval-Augmented Generation (RAG) system using LM Studio and Python.

## Features
- PDF Question Answering using embeddings
- CSV data querying
- Local LLM inference using LM Studio

## Project Structure

Lm_studio
│
├── data
│   └── Stu_Data.xlsx
├── requirements.txt
└── src
    ├── csv_rag.py
    └── pdf_rag_ram.py

## Installation

Install dependencies:

pip install -r requirements.txt

## Run Example

python src/pdf_rag_ram.py

## Tools Used
- Python
- LM Studio
- Embeddings
- Vector Search
